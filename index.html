📋 Cómo configurar Google Sheets
1. Abre Google Sheets y crea una hoja con columnas:
  Fecha | Sentimiento | Comentario | Contexto

2. En el menú: Extensiones → Apps Script

3. Pega este código:
function doPost(e) {
  var sheet = SpreadsheetApp
    .getActiveSpreadsheet()
    .getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    new Date(),
    data.sentiment,
    data.comment || '',
    data.context || ''
  ]);
  return ContentService
    .createTextOutput(
      JSON.stringify({status:'ok'})
    )
    .setMimeType(
      ContentService.MimeType.JSON
    );
}
4. Despliega como Web App → acceso Cualquier usuario

5. Copia la URL y pégala arriba ↑
